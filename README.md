# Agriculture - Project

Capstone project for Machine Learning Engineering . 
Project: Predicting Crops Yield : A Machine Learning Approach
Problem: Predict crops yield for 10 of most consumed crops worldwide.

Environment
OS: Mac OS
Python : 3
Anaconda 64 bit 

Install
This project requires Python and the following Python libraries installed:

NumPy
Pandas
matplotlib
scikit-learn 
seaborn
pydot
graphviz

Code is provided in the prediction_model_yield.ipynb. 


Dataset:
http://www.fao.org/home/en/  
https://data.worldbank.org/
